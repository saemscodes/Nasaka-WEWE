# Nasaka WEWE

Nasaka WEWE is a secure messaging app built for Kenyan activists, journalists, civic educators and anyone who needs a safe, resilient way to communicate — even during internet blackouts.

Forked from the Briar Project's battle-tested encrypted messaging engine, Nasaka WEWE preserves full end-to-end encryption while adding CEKA (Civic Education Kenya) membership integration and an offline-first authentication flow.

Unlike traditional messaging apps, Nasaka WEWE doesn't rely on a central server — messages are synchronized directly between users' devices.

If the Internet's down, Nasaka WEWE can sync via Bluetooth, Wi-Fi or memory cards, keeping information flowing in a crisis. If the Internet's up, Nasaka WEWE can sync via the Tor network, protecting users and their relationships from surveillance.

## Features

- **Private Messages** — End-to-end encrypted one-on-one conversations
- **Private Groups** — Invite-only group messaging
- **Forums** — Open discussion spaces shared with contacts
- **Blogs** — Publish and subscribe to community updates
- **Offline Sync** — Bluetooth, Wi-Fi and removable drive support
- **Tor Integration** — Built-in Tor for surveillance-resistant internet connections
- **Dual-Mode Auth** — CEKA online authentication or offline local account creation
- **No Ads, No Tracking** — Your data stays on your device
